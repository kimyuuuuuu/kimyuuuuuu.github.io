## 파일 & 디렉토리

### ls — 목록 보기

| 명령어/플래그 | 설명 |
|---|---|
| `ls` | 기본 목록 |
| `ls -l` | 상세 정보 (권한, 크기, 날짜) |
| `ls -a` | 숨김 파일 포함 |
| `ls -la` | 상세 + 숨김 파일 |
| `ls -lh` | 파일 크기를 KB/MB로 표시 |
| `ls -lt` | 최근 수정 순 정렬 |
| `ls -ltr` | 오래된 순 정렬 |
| `ls -R` | 하위 디렉토리까지 재귀적으로 |

### cd — 디렉토리 이동

| 명령어/플래그 | 설명 |
|---|---|
| `cd [경로]` | 지정한 디렉토리로 이동 |
| `cd ..` | 상위 디렉토리로 |
| `cd ~` | 홈 디렉토리로 |
| `cd -` | 이전 디렉토리로 |
| `cd /` | 루트 디렉토리로 |

### pwd — 현재 경로 확인

| 명령어/플래그 | 설명 |
|---|---|
| `pwd` | 현재 작업 디렉토리 출력 |

### mkdir — 디렉토리 만들기

| 명령어/플래그 | 설명 |
|---|---|
| `mkdir [디렉토리명]` | 디렉토리 생성 |
| `mkdir -p a/b/c` | 중간 디렉토리도 자동 생성 |
| `mkdir -m 755 [디렉토리명]` | 권한까지 지정해서 생성 |

### rm — 삭제

| 명령어/플래그 | 설명 |
|---|---|
| `rm [파일]` | 파일 삭제 |
| `rm -r [디렉토리]` | 디렉토리 통째로 삭제 |
| `rm -f [파일]` | 확인 없이 강제 삭제 |
| `rm -rf [디렉토리]` | 강제로 디렉토리 통째 삭제 |
| `rm -i [파일]` | 삭제 전 확인 |

### cp — 복사

| 명령어/플래그 | 설명 |
|---|---|
| `cp [원본] [대상]` | 파일 복사 |
| `cp -r [디렉토리1] [디렉토리2]` | 디렉토리 복사 |
| `cp -p [파일1] [파일2]` | 권한/날짜 속성 유지하며 복사 |
| `cp -i [원본] [대상]` | 덮어쓰기 전 확인 |

### mv — 이동 / 이름 바꾸기

| 명령어/플래그 | 설명 |
|---|---|
| `mv [원본] [대상]` | 파일 이동 또는 이름 변경 |
| `mv -i [원본] [대상]` | 덮어쓰기 전 확인 |

## 검색

### grep — 텍스트 검색

| 명령어/플래그 | 설명 |
|---|---|
| `grep [패턴] [파일]` | 파일에서 패턴 검색 |
| `grep -i [패턴] [파일]` | 대소문자 구분 없이 |
| `grep -r [패턴] [디렉토리]` | 디렉토리 내 전체 검색 |
| `grep -n [패턴] [파일]` | 줄 번호도 표시 |
| `grep -v [패턴] [파일]` | 패턴이 없는 줄만 출력 |
| `grep -l [패턴] [파일들]` | 파일명만 출력 |
| `grep -c [패턴] [파일]` | 매칭된 줄 개수 |

### find — 파일 찾기

| 명령어/플래그 | 설명 |
|---|---|
| `find [경로] -name [패턴]` | 이름으로 찾기 |
| `find [경로] -type d` | 디렉토리만 |
| `find [경로] -type f` | 파일만 |
| `find [경로] -mtime -7` | 최근 7일 내 수정된 파일 |
| `find [경로] -size +10M` | 10MB 이상 파일 |
| `find [경로] -user [사용자]` | 특정 사용자 소유 파일 |


## 시스템 & 프로세스

### ps — 프로세스 목록

| 명령어/플래그 | 설명 |
|---|---|
| `ps` | 현재 터미널의 프로세스 |
| `ps aux` | 모든 프로세스 상세 정보 |
| `ps -ef` | 모든 프로세스 (다른 형식) |
| `ps -u [사용자]` | 특정 사용자의 프로세스 |

### top / htop — 실시간 모니터

| 명령어/플래그 | 설명 |
|---|---|
| `top` | 실시간 시스템 모니터 |
| `top -u [사용자]` | 특정 사용자 프로세스만 |
| `htop` | 개선된 인터페이스 (설치 필요) |

### kill — 프로세스 종료

| 명령어/플래그 | 설명 |
|---|---|
| `kill [PID]` | 프로세스 종료 |
| `kill -9 [PID]` | 강제 종료 |
| `kill -15 [PID]` | 정상 종료 (기본값) |
| `killall [프로세스명]` | 이름으로 종료 |

### nice / renice — 우선순위 조정

| 명령어/플래그 | 설명 |
|---|---|
| `nice -n [값] [명령어]` | 낮은 우선순위로 실행 (-20~19) |
| `renice -n [값] -p [PID]` | 실행 중 우선순위 변경 |

### jobs / fg / bg — 작업 제어

| 명령어/플래그 | 설명 |
|---|---|
| `jobs` | 백그라운드 작업 목록 |
| `fg %[번호]` | 포그라운드로 전환 |
| `bg %[번호]` | 백그라운드로 전환 |
| `[명령어] &` | 백그라운드로 실행 |

### lsof — 열린 파일 목록

| 명령어/플래그 | 설명 |
|---|---|
| `lsof` | 모든 열린 파일 |
| `lsof -p [PID]` | 특정 프로세스가 연 파일 |
| `lsof -i :[포트]` | 특정 포트 사용 프로세스 |
| `lsof -u [사용자]` | 특정 유저가 연 파일 |


## 네트워크

### ping — 연결 테스트

| 명령어/플래그 | 설명 |
|---|---|
| `ping [호스트]` | 연결 확인 |
| `ping -c [횟수] [호스트]` | 지정 횟수만 핑 |

### curl — HTTP 요청

| 명령어/플래그 | 설명 |
|---|---|
| `curl [URL]` | URL 내용 가져오기 |
| `curl -O [URL]` | 파일 다운로드 |
| `curl -o [파일명] [URL]` | 파일명 지정하여 다운로드 |
| `curl -I [URL]` | 헤더만 보기 |
| `curl -L [URL]` | 리다이렉트 따라가기 |
| `curl -X POST [URL]` | POST 요청 |
| `curl -u [user:pass] [URL]` | 인증 |

### wget — 파일 다운로드

| 명령어/플래그 | 설명 |
|---|---|
| `wget [URL]` | 파일 다운로드 |
| `wget -c [URL]` | 중단된 다운로드 재개 |
| `wget -r [URL]` | 재귀적으로 다운로드 |

### ssh — 원격 접속

| 명령어/플래그 | 설명 |
|---|---|
| `ssh [user@host]` | 원격 서버 접속 |
| `ssh -p [포트] [user@host]` | 포트 지정 |
| `ssh -i [키파일] [user@host]` | 키 파일로 접속 |

### scp — 원격 파일 복사

| 명령어/플래그 | 설명 |
|---|---|
| `scp [파일] [user@host:경로]` | 서버로 업로드 |
| `scp [user@host:경로] [로컬경로]` | 서버에서 다운로드 |
| `scp -r [디렉토리] [user@host:경로]` | 디렉토리 복사 |
| `scp -P [포트] [파일] [user@host:경로]` | 포트 지정 |

### ss / netstat — 네트워크 상태

| 명령어/플래그 | 설명 |
|---|---|
| `ss -tuln` | 열려있는 포트 확인 |
| `ss -tp` | TCP 연결 + 프로세스 |
| `netstat -tuln` | ss와 동일 (구버전) |

### dig / nslookup — DNS 조회

|---|---|

| `dig [도메인] MX` | 메일 서버 조회 |
| `dig @[DNS서버] [도메인]` | 특정 DNS 서버로 조회 |
| `nslookup [도메인]` | 간단한 DNS 조회 |

### ip — 네트워크 인터페이스

| 명령어/플래그 | 설명 |
|---|---|
| `ip addr` | IP 주소 확인 |
| `ip route` | 라우팅 테이블 |
| `ip link` | 네트워크 인터페이스 목록 |


## 권한 관리

### chmod — 권한 변경

| 명령어/플래그 | 설명 |
|---|---|
| `chmod [권한] [파일]` | 권한 변경 |

| `chmod -w [파일]` | 쓰기 권한 제거 |
| `chmod -R [권한] [디렉토리]` | 재귀적으로 적용 |

### chown — 소유자 변경

| 명령어/플래그 | 설명 |
|---|---|
| `chown [user] [파일]` | 소유자 변경 |
| `chown [user:group] [파일]` | 소유자 + 그룹 변경 |
| `chown -R [user] [디렉토리]` | 재귀적으로 적용 |


## 디스크 & 파일시스템

### df — 디스크 용량

| 명령어/플래그 | 설명 |
|---|---|
| `df` | 모든 파티션 용량 |
| `df -h` | 사람이 읽기 쉽게 (KB/MB/GB) |

| `df -T` | 파일시스템 타입 표시 |

| `df -t [타입]` | 특정 파일시스템만 |
| `df -x [타입]` | 특정 타입 제외 |
| `df --total` | 합계 표시 |

### du — 디렉토리/파일 용량

| 명령어/플래그 | 설명 |
|---|---|
| `du` | 현재 디렉토리 용량 |

### lsblk / mount — 디스크 관리

| 명령어/플래그 | 설명 |
|---|---|
| `lsblk` | 연결된 디스크/파티션 목록 |

## 압축 & 아카이브

| 명령어/플래그 | 설명 |
|---|---|
| `tar -cvf [파일.tar] [디렉토리]` | 묶기만 |
| `tar -czvf [파일.tar.gz] [디렉토리]` | gzip 압축 |
| `tar -cjvf [파일.tar.bz2] [디렉토리]` | bzip2 압축 |
| `tar -xzvf [파일.tar.gz]` | 압축 해제 |

| `tar -tzvf [파일.tar.gz]` | 내용 확인만 |
| `-c` | 만들기 (create) |
| `-x` | 풀기 (extract) |
| `-t` | 목록보기 (list) |
| `-v` | 진행상황 출력 |
| `-z` | gzip |
| `-j` | bzip2 |
| `-f` | 파일 지정 |

### zip / unzip

## 텍스트 처리

### sed — 텍스트 치환

| 명령어/플래그 | 설명 |
|---|---|
| `sed 's/old/new/' [파일]` | 첫 번째만 치환 |
| `sed 's/old/new/g' [파일]` | 전체 치환 |
| `sed -i 's/old/new/g' [파일]` | 파일 직접 수정 |
| `sed -n '5,10p' [파일]` | 5~10번째 줄만 출력 |
| `sed '/패턴/d' [파일]` | 패턴 있는 줄 삭제 |

### awk — 컬럼 기반 처리

| 명령어/플래그 | 설명 |
|---|---|
| `awk '{print $1}' [파일]` | 첫 번째 컬럼만 |
| `awk '{print $1, $3}' [파일]` | 1번, 3번 컬럼 |
| `awk -F: '{print $1}' [파일]` | 구분자 지정 |
| `awk 'NR==[숫자]' [파일]` | 특정 줄만 |

### sort — 정렬

| 명령어/플래그 | 설명 |
|---|---|
| `sort [파일]` | 알파벳 순 정렬 |
| `sort -r [파일]` | 역순 |
| `sort -n [파일]` | 숫자 순 |
| `sort -k[번호] [파일]` | 특정 컬럼 기준 |
| `sort -u [파일]` | 중복 제거하며 정렬 |
| `sort -h [파일]` | 사람이 읽기 쉬운 숫자 (1K, 2M) |

### uniq — 중복 처리

| 명령어/플래그 | 설명 |
|---|---|
| `uniq [파일]` | 연속 중복 제거 |
| `uniq -c [파일]` | 중복 횟수 표시 |
| `uniq -d [파일]` | 중복된 것만 출력 |

### wc — 글자/줄/단어 세기

| 명령어/플래그 | 설명 |
|---|---|
| `wc [파일]` | 줄/단어/바이트 수 |
| `wc -l [파일]` | 줄 수만 |
| `wc -w [파일]` | 단어 수만 |
| `wc -c [파일]` | 바이트 수만 |

| 명령어/플래그 | 설명 |
|---|---|

### tr — 문자 변환

### diff — 파일 비교

| 명령어/플래그 | 설명 |
|---|---|

## 파이프 & 리다이렉션

| 기호 | 설명 |

| `>>` | 출력을 파일에 추가 |

| `2>&1` | 에러를 표준 출력으로 |
| `&` | 백그라운드 실행 |
| `&&` | 앞 성공 시 다음 실행 |
| `&#124;&#124;` | 앞 실패 시 다음 실행 |
